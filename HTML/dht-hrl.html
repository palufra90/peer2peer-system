Record Definitions

job

-record(job, {time::string(), node::node(), module::atom(), function::atom(), args::[atom()], term()}).

time: timestamp of the moment when the job is inserted in DHt
node: node that inserted the job in DHT
module: name of the module of the function to execute
function: name of function to execute
args: args of the function to execute
result: result of the job

code

-record(code, {module::atom(), binary::binary()}).

module: name of the module of the function to execute
binary: binary of the object code of the function to execute

candidates

-record(candidates, {job::binary(), nodes::[node(), int()]}).

job: key of the job
nodes: list of the candidates with respective node name and performance

myJobs

-record(myJobs, {node::node(), jobs::[binary()]}).

node: node name
jobs: list of keys of the jobs added from the node
